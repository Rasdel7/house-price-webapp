# House Price Prediction Web App 🏠

Live web app that predicts California house prices using
Random Forest ML model.

## Live Demo
[Click here to try the app](YOUR_STREAMLIT_URL_HERE)

## Tools Used
- Python, Scikit-learn, Streamlit, Matplotlib, Pandas

## Features
- Interactive sliders to adjust house parameters
- Real-time price prediction
- Feature importance visualization
- Clean dashboard layout

## How to Run Locally
pip install streamlit scikit-learn matplotlib pandas
streamlit run app.py
